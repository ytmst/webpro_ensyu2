演習課題3

　

この記事では、HTMLの数ある要素のうち、基本的な3種類の要素を紹介します。

リスト

リストは<ul></ul>タグ,もしくは<ol></ol>用いて実装します。
<ul>タグは順序なしのリスト、<ol>タグは順序ありのリストです。
<ul>、もしくは<ol>要素でリストを囲み、箇条書きしたい項目を<li>タグで囲むことで追加できます。 ここでは、<ul>タグを用いてリストを作成してみました。 コードは以下の通りです。


          <ul> 
            <li>あいうえお</li>
            <li>かきくけこ</li>
            <li>さしすせそ</li>
          </ui>

実行結果は以下の通りです。

  • あいうえお
  • かきくけこ
  • さしすせそ

リストの点を変更することもできます。例えば、


          ul {
            list-style-type: circle;
          }

として、黒丸を白丸に変更してみました。以下のようになります。

  • リスト1
  • リスト2
  • リスト3

<ol>タグを用いた順序付きリストも同様に作成できます。ここでは例は省略。

表

<table>タグを用いて表を作成できます。<th>,<tr>,<td>要素を用いて作ります。

コードは以下の通りです。


          <table>
          <tr>
            <th>タグ</th>
            <th>説明</th>
          </tr>
          <tr>
            <td><tr></td>
            <td>表の行を囲む。
            <strong>t</strong>able <strong>r</strong>ow の略。
            </td>
          </tr>
          <tr>
            <td><tr></td>
            <td>
            表の見出しとなるセルを作成する。
            <strong>t</strong>able <strong>h</strong>eader の略。
            </td>
          </tr>
          <tr>
            <td><tr></td>
            <td>
            表のデータとなるセルを作成する。
            <strong>t</strong>able <strong>d</strong>eta の略。
            </td>
          </tr>
        </table>

完成した表がこちらです。この表で各タグについて説明します。

タグ 説明
<tr> 表の行を囲む。table row の略。
<th> 表の見出しとなるセルを作成する。table header の略。
<td> 表のデータとなるセルを作成する。table data の略。

画像

最後は画像の挿入について学びましょう。画像を表示するには<img>タグを使用します。コードは以下のように書きます。


          <img src = "images/sunflower.jpg" alt = "ひまわり">

src属性は場所を設定する属性です。ここでは、画像ファイルがある場所を指定しています。src属性にファイルのパスを書き込むことで画像が表示できるという仕組みになっています。 次に、コードの末尾についている<alt>タグに注目してください。これは、もし何らかの問題でブラウザで画像を読み込めなかった場合に、画像の代わりに表示するテキストです。 また、テキストは表示されていない画像がどんな画像なのかを書くべきです。 実行結果はこちらです。
